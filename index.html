📋 Admin Roles Guide

Choose the right role for each team member. You can assign multiple roles to combine access.

Super Admin
Full system access including managing admin accounts. Only one super admin exists.
Access: Everything
Admin (full)
Full operational access to all platform features except managing other admins.
Access: All sections except Admin Accounts
Bookings & Fees
Manage all bookings, lifecycle, archive, analytics, platform fees, and professional applications.
Access: Dashboard, Bookings, Lifecycle, Archive, Analytics, Platform Fees, Applications
Best for: Operations manager handling bookings and revenue
Locations
Manage countries, governorates, and cities. Perfect for expansion team.
Access: Dashboard, Locations only
Best for: Geographic expansion specialist
Tickets
Handle support tickets, respond to users, close resolved issues.
Access: Dashboard, Tickets only
Best for: Customer support agent (tickets only)
Categories & Media
Manage service categories, upload images, bulk operations.
Access: Dashboard, Categories only
Best for: Content manager for service catalog
Users
Manage user accounts, ban/unban users, view user details.
Access: Dashboard, Users only
Best for: Community moderator
Platform Fees
Confirm payments, release suspensions, view financial dashboard.
Access: Dashboard, Platform Fees only
Best for: Finance/accounting team member
Notifications
Send push notifications to users, professionals, or specific individuals.
Access: Dashboard, Push Notifications only
Best for: Marketing/communications specialist
Support Lead
Handle tickets and manage users. Combined role for senior support staff.
Access: Dashboard, Tickets, Users
Best for: Senior support lead managing both tickets and user issues
Verifier
Review and approve professional applications only.
Access: Dashboard, Applications only
Best for: Application reviewer (verifying professional sign-ups)

Clear recorded platform fees

This resets recorded fee counters on every professional and branch, removes stored platform-fee fields on booking matches, deletes all payment confirmation rows (platform_fee_payments), clears monthly clearances, and releases fee suspensions. Use only for testing — you will lose admin payment history in the panel.

Type the phrase below exactly to confirm:

RESET_ALL_RECORDED_FEES
